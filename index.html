# 快乐老家画室

# 1.画室简介

  • 哈喽大家好，这里是最终幻想14猫小胖区-快乐老家速写室，欢迎大家参与速写室活动，让自己的角色拥有一个属于自己的头像，让热爱FF14的玩家得到一个快乐的社区！

  • 画室活动策划均由静语庄园的热爱者用爱发电，希望大家多多包容，也希望能和大家共同友好相处，快乐度过每个周末的活动，群号：173514525

# 2.活动范围

  • 目前画室活动基本在周六周日，具体时间会在活动时间表中提醒

  • 每周六周日随机开放！！！进入房屋默认同意被画/随便画！欢迎大家一起挂机画速写清稿子！快乐练习画画！

# 3.活动内容

  • 想要喜欢或者热爱画画的人一起突破舒适区！一起进步！一起画画！

# 4.画师资格的审核和申请

  • 如有画师想申请画师资格一起快乐画画，可以进群联系画室管理员✔️

  • 每次活动的参与非硬性要求✔️
